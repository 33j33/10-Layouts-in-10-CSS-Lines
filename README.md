# 10-Layouts-in-10-CSS-Lines
Minimal Modern CSS layouts in just 10 lines of CSS

## 1. Centered 

```css
.container {
    display: grid;
    place-items: center;
}
```
Centering is one of the most challenging task in CSS. This centers everything perfectly (both horizontally and vertically)
